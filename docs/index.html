自分でBotを設定する

GitHubからコードを保存し、自分でbotを設定することも可能です。

  1. GitHubからコピー： 
    git clone https://github.com/fso2729/Anikibot.git
  2. 必要データをインストール： 
    npm install
  3. .envを作成してAPIキーを設定
  4. Botを起動： 
    node index.js

コマンド一覧

  • !ask - 質問
  • !chat - 雑談
  • !search - 検索
  • !reset - 会話履歴リセット
  • !help - ヘルプ表示
